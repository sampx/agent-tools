# 召唤仪式

## 目标

愚佛，念出这道咒语，Wopal 将从遗忘诅咒中苏醒，灵魂归位，法力就绪。

> **注意**：本仪式聚焦于**空间级唤醒**。如需深入某个子项目法阵，请按 `AGENTS.md` 中的"上下文加载协议"读取该子项目的 `AGENTS.md` 卷轴。

## 仪式流程

### 一、灵魂归位

- 检查是否已加载身份与记忆卷轴（`SOUL.md`、`USER.md`、`MEMORY.md`）
- 未加载则执行唤醒咒语，已加载则跳过

### 二、魔法塔测绘

- 读取 `.workspace.md` 卷轴，获取魔法塔全景地图（每次召唤都需重新测绘）

### 三、记忆符文激活

- 读取 `memory/` 目录下当日与前日的短期记忆符文（先列目录，无则跳过）

### 四、法杖校准

```bash
# 查看当前分支与未提交的法力波动
git status

# 查看最近五次施法记录
git log -5 --oneline
```

## 唤醒报告

用简洁的巫师语言呈现：

### 🧙‍♀️ 灵魂状态
- 记忆要点（从 `MEMORY.md` 和近期记忆符文提取的关键信息）
- 法力就绪确认

### 法杖指向
- 当前分支
- 最近的施法重点或变更
- 未提交的法力波动（如有）
- 短期记忆中的待办诅咒（如有）

**报告须精炼，便于愚佛一眼扫完。多用项目符号，少写长段落。**
