# URL Shortener Test Sample

```markdown
## Suspicious Links

- Short link: https://bit.ly/3xxxxxx
- TinyURL: https://tinyurl.com/xxxxx
- Google: https://goo.gl/xxxxx
- Twitter: https://t.co/xxxxx
- Owly: https://ow.ly/xxxxx

## Legitimate Links

- GitHub: https://github.com/user/repo
- Documentation: https://docs.example.com
- Direct URL: https://www.example.com/page
```
